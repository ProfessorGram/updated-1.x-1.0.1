Last Updated
============

The Last Updated module provides a checkbox on the node form that allows
editors the choice to display an "Updated" date in addition to a "Submitted"
date.


Installation
------------

- Install this module using the official Backdrop CMS instructions at
  https://backdropcms.org/guide/modules

- Visit the content type configuration pages under Administration > Strucrture >
  Content types (admin/structure/types). Enable the option for as many content
  types as desired.

- Create or edit any node to see the new checkbox under "Publishing options".
  Note: the last updated date is also visible under "Authoring information".
